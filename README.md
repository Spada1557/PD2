# Проектная деятельность 2

Тема проекта: **Разработка информационной системы предварительного заказа блюд в студенческой столовой**.

Репозиторий содержит рабочий прототип информационной системы и материалы проектной документации.

## Структура

```text
app/            статическое Vue-приложение
docs/           отчет, спецификации, диаграммы и проектные материалы
tools/          скрипты для генерации документации
```

## Запуск прототипа

Откройте файл `app/index.html` в браузере.

Демо-доступ сотрудника столовой:

```text
логин: admin
пароль: admin123
```

Обычный пользователь может зарегистрироваться через форму входа.

## Документация

Исходные материалы отчета находятся в `docs/report`. Черновой DOCX формируется командой:

```bash
python3 tools/build_report.py
```

Файл результата: `docs/report/project_report.docx`.

Логотип и цветовая схема описаны в `docs/specs/brand_guidelines.md`.
